# ZackType Icons

Place the following icon files in this directory before loading the extension:

- `icon16.png` — 16×16 px
- `icon32.png` — 32×32 px
- `icon48.png` — 48×48 px
- `icon128.png` — 128×128 px

You can use any image editor or online tool to generate these icons.
A simple blue "Z" on a dark background works well to match the ZackType branding.
